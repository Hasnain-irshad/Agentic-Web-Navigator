"""
QUICK START GUIDE - Agentic AI Web Navigator

Follow these steps to get started.
"""

# ==============================================================================
# STEP 1: PREREQUISITES
# ==============================================================================

Before you begin:
  ✓ Python 3.9 or higher
  ✓ pip or conda package manager
  ✓ ~500MB disk space (for Playwright browsers)
  ✓ Internet connection (for Groq API, Playwright browsers)

Get your Groq API key (free):
  1. Visit https://console.groq.com
  2. Sign up or log in
  3. Go to API Keys section
  4. Create a new API key
  5. Copy the key (you'll need it in Step 4)

# ==============================================================================
# STEP 2: SETUP PYTHON ENVIRONMENT
# ==============================================================================

In your terminal/PowerShell, run:

# Create virtual environment
python -m venv venv

# Activate it (choose based on your OS):
# On Windows:
venv\Scripts\activate

# On macOS/Linux:
source venv/bin/activate

# You should see (venv) in your terminal prompt

# ==============================================================================
# STEP 3: INSTALL DEPENDENCIES
# ==============================================================================

# Install Python packages
pip install -r requirements.txt

# Install Playwright browsers (required)
playwright install chromium

# This downloads the browser binary (~200MB)
# Be patient, it may take a few minutes

# ==============================================================================
# STEP 4: CONFIGURE API KEY
# ==============================================================================

# Copy the environment template
cp .env.template .env

# Edit .env file and add your Groq API key:
# GROQ_API_KEY=your_actual_key_from_console.groq.com

# Other options you can configure:
# LLM_MODEL=llama-3.1-8b-instant
# LLM_TEMPERATURE=0.1
# HEADLESS=true  (set to false to see browser)
# MAX_STEPS=20

# ==============================================================================
# STEP 5: VERIFY INSTALLATION
# ==============================================================================

# Run the quickstart validator
python quickstart.py

# This checks:
  ✓ All packages installed
  ✓ Configuration valid
  ✓ Project structure correct
  ✓ Agent can initialize

# If all checks pass, you're ready!

# ==============================================================================
# STEP 6: TRY YOUR FIRST TASK
# ==============================================================================

EASIEST: Use the GUI Interface
  python gui.py
  
  ✓ Graphical interface (point and click)
  ✓ Real-time execution log
  ✓ Task summary with metrics
  ✓ Save results to JSON
  ✓ Mock mode for testing without API key
  
  See GUI_GUIDE.txt for detailed instructions.

ALTERNATIVE: Use Command Line
  
  Option A: Test with mock (no API key needed)
  python main.py --goal "Test navigation" --mock --visible
  
  Option B: Real task (requires Groq API key)
  python main.py --goal "Search for Python on Google" --visible
  
  Option C: Run example tasks
  python examples/sample_tasks.py

# ==============================================================================
# STEP 7: UNDERSTAND THE OUTPUT
# ==============================================================================

When you run a task, you'll see:

  OBSERVE → What the agent sees on the page
  REASON  → LLM decides what to do next
  EXECUTE → Browser performs the action
  STORE   → Memory records the action

Example output:
  ✓ Step 1: GOTO → Navigated to https://google.com
  ✓ Step 2: CLICK → Clicked on Search
  ✓ Step 3: TYPE → Typed "Python"
  ✓ Step 4: CLICK → Clicked Search Button
  ✓ Step 5: DONE → Task completed

Final summary shows:
  - Goal achieved?
  - Total steps taken
  - Successful vs failed actions
  - Full action sequence

# ==============================================================================
# PYTHON API USAGE
# ==============================================================================

from main import WebNavigatorAgent
import asyncio

async def my_task():
    agent = WebNavigatorAgent(
        goal="Find something on the web",
        max_steps=15,
        headless=False,  # Show browser
        use_mock=False,   # Use real LLM
    )
    result = await agent.run()
    
    print(f"Status: {result['status']}")
    print(f"Steps: {result['total_steps']}")
    
    return result

asyncio.run(my_task())

# ==============================================================================
# TROUBLESHOOTING
# ==============================================================================

Problem: "GROQ_API_KEY not set"
Solution:
  1. Create .env file (copy from .env.template)
  2. Add your Groq API key
  3. Restart terminal/Python

Problem: "playwright install failed"
Solution:
  1. Check internet connection
  2. Run: playwright install --with-deps
  3. Try again

Problem: "No elements found on page"
Solution:
  1. Run with --visible to see page
  2. Page might need scroll
  3. Try simpler task first

Problem: "Browser hangs/timeout"
Solution:
  1. Press Ctrl+C to stop
  2. Check BROWSER_TIMEOUT in .env
  3. Try simpler page (example.com)

Problem: "API rate limited"
Solution:
  1. Wait a few minutes
  2. Groq free tier has limits
  3. Try --mock mode for testing
