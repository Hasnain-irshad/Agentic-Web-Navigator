WHAT WAS IMPLEMENTED
================================================================================

Two generic, website-agnostic mechanisms to prevent infinite loops:

(1) DETERMINISTIC COMPLETION DETECTOR
   - Detects when goal is achieved using generic signals
   - Works across ANY website (no hardcoded domains)
   - Signals: result keywords, result elements, URL changes, goal keywords
   - Examples: search results, navigation success, generic achievement

(2) ANTI-REPEAT GUARD
   - Tracks action signatures (type, selector, value, key)
   - Detects repetition (same action 3+ times)
   - Detects no-progress (page state unchanged)
   - Stops early with explanation message

USAGE EXAMPLE
================================================================================

# No configuration needed!
from core.session_agent import SessionAgent

agent = SessionAgent(max_steps_per_command=20)
await agent.start_session()

# Automatic stop conditions work out-of-box
result = await agent.execute_command("search for laptop on daraz")

# Agent stops automatically when:
# - Search results detected, OR
# - Same action repeated 3x, OR  
# - No page change 2+ steps, OR
# - LLM returns DONE action

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

"Agent keeps repeating the same action"
→ Check stop_reason in result dict
→ May need to increase max_steps_per_command
→ Verify page is actually changing

"Agent stops too early"
→ Check observation.page_text has enough content
→ May be detecting false positive completion
→ Verify goal keywords don't match random page text

"RepetitionDetector not triggering"
→ Action signatures must be identical (type, selector, value, key)
→ Page state must be identical (URL + title)
→ May need more steps for detection

"Getting import errors"
→ Verify core/__init__.py is not blocking stop_conditions
→ Check Python version (3.8+ required)
→ Ensure all dependencies installed

See PATCH_GUIDE.md section "Troubleshooting" for more details.
