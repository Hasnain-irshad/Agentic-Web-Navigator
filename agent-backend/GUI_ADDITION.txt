NEW FILE: gui.py (450 lines)
─────────────────────────────────────────────────────────────

A complete Tkinter-based graphical user interface for testing and running
the Agentic Web Navigator without using the command line.

FEATURES:

1. Task Input Section
   • Multi-line text area for entering task goals
   • Pre-filled with example task
   • Easy to modify and re-run

2. Configuration Options
   • Headless mode toggle (ON/OFF)
     - ON: Fast, hidden browser
     - OFF: Visible browser window
   
   • Mock mode toggle (ON/OFF)
     - ON: Testing without API calls
     - OFF: Real LLM reasoning
   
   • Max steps spinner (1-100)
     - Adjustable for task complexity
     - Default: 20

3. Execution Controls
   • START TASK button - Begin execution
   • STOP button - Interrupt running task
   • SAVE RESULT button - Export to JSON
   • CLEAR LOG button - Clear output

4. Real-Time Execution Log
   • Timestamped messages
   • Color-coded output:
     - Blue: Information
     - Green: Success/completed
     - Red: Errors/failures
     - Orange: Warnings/status changes
     - Purple: Step indicators
   • Auto-scrolling to latest
   • Monospace font for readability

5. Task Summary Panel
   • Status display (COMPLETED/FAILED/ERROR)
   • Metrics:
     - Steps taken / max steps
     - Successful actions count
     - Failed actions count
   • Action sequence (first 10 shown)
   • Color-coded success/failure

6. Status Indicator
   • Real-time status display
   • Color changes with status
   • Shows: READY, RUNNING, COMPLETED, ERROR, STOPPED

7. Save Functionality
   • Export complete results to JSON
   • Auto-generated filename with timestamp
   • Includes all task metadata
   • Easy file browser dialog

TECHNICAL DETAILS:

• Uses: Tkinter (built-in to Python)
• No additional dependencies needed
• Threading for non-blocking UI
• Async integration with main agent
• Proper error handling
• Clean shutdown

# ==============================================================================
# HOW TO USE THE GUI
# ==============================================================================

LAUNCH:
  python gui.py

BASIC WORKFLOW:
  1. Enter task goal in text area
  2. Adjust options if needed (defaults are good)
  3. Click START TASK
  4. Watch real-time execution log
  5. View summary when complete
  6. Click SAVE RESULT to export

EXAMPLE TASKS:
  
  Simple: "Navigate to Wikipedia"
  
  Moderate: "Search for Python on Google"
  
  Complex: "Find the most starred Python repo on GitHub"
