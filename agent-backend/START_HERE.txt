A COMPLETE AUTONOMOUS WEB NAVIGATION SYSTEM

The agent can:
  ✓ Navigate websites (GOTO, BACK, tabs)
  ✓ Interact with pages (CLICK, TYPE, SCROLL)
  ✓ Extract visible elements (no HTML parsing)
  ✓ Reason using LLM (Groq Llama models)

  ✓ Generate task summaries
  ✓ Self-correct on failures
  ✓ Enforce step limits

All within hard constraints:
  ✓ No authentication or CAPTCHAs
  ✓ No payments or transactions
  ✓ No hardcoded flows (pure LLM-driven)
  ✓ No BeautifulSoup (Playwright only)
  ✓ Strict one-action-per-step schema

# HOW TO GET STARTED

STEP 1: INSTALL
  pip install -r requirements.txt
  playwright install chromium

STEP 2: CONFIGURE
  cp .env.template .env
  # Edit .env and add your Groq API key

  python quickstart.py

STEP 4: TRY IT
  # Without API key (mock mode):
  python main.py --goal "Navigate to Google" --mock --visible
  
  # With API key (real LLM):
  python main.py --goal "Search for Python on Google" --visible
