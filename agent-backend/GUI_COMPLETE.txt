# ==============================================================================

LOG MESSAGE COLORS:

Blue    [ℹ] Information
        ├─ Task started/completed
        ├─ Mode information
        ├─ Goal description
        └─ General messages

Green   [✓] Success
        ├─ Action completed
        ├─ Task finished
        ├─ Result saved
        └─ Positive outcomes

Red     [✗] Error/Failure
        ├─ Action failed
        ├─ Task failed
        ├─ API errors
        └─ Critical issues

Orange  [⚠] Warning/Status
        ├─ Step changes
        ├─ Mode changes
        ├─ Task stopped
        └─ Status updates

Purple  [→] Step Indicators
        ├─ Step N: ACTION
        ├─ Action names
        └─ Progress markers

CYAN    [→] Action Details
        ├─ Action parameters
        ├─ Element selectors
        └─ Detailed info

SUMMARY COLORS:

Green   ✓ Successful status/actions
Red     ✗ Failed status/actions
Default  Neutral information

# ==============================================================================
# TROUBLESHOOTING
# ==============================================================================

GUI Won't Start:
  ✓ Ensure Python 3.9+
  ✓ Check Tkinter: python -m tkinter
  ✓ Linux: sudo apt-get install python3-tk

GUI Freezes:
  ✓ This is normal during task execution
  ✓ Task runs in background thread
  ✓ Log updates in real-time
  ✓ Don't click buttons during execution

No Log Output:
  ✓ Check if task is running
  ✓ Status should show RUNNING
  ✓ Log updates every few seconds
  ✓ May be slow if network is slow

Task Won't Run:
  ✓ Check goal is entered
  ✓ Check API key if Mock is OFF
  ✓ Or enable Mock Mode
  ✓ Try simpler task first

Can't Find Save Dialog:
  ✓ Dialog may appear behind window
  ✓ Alt+Tab to switch windows
  ✓ Or try Save again
