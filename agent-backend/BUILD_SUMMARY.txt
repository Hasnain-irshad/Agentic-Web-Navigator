✓ No CAPTCHA handling  - Skips CAPTCHA pages (returns failure)

✓ No HTML parsing      - Uses Playwright locators only

✓ Deterministic        - Same input produces predictable behavior

# ==============================================================================
# PERFORMANCE NOTES
# ==============================================================================

Typical Task Timeline
  ├─ Step 1: Navigate to Google (2-3s)
  ├─ Extract observation (500ms)
  ├─ LLM reasoning (3-5s)
  ├─ Execute action (500ms-2s)
  └─ Repeat ~5-20 steps

Total Task Time
  ├─ Simple search: ~1-2 minutes
  ├─ Complex navigation: ~5-10 minutes
  └─ Multiple clicks: ~20-30 steps

Bottlenecks
  1. LLM API latency (~3-5s per step)
  2. Page load times (depends on site)
  3. Element extraction (scales with DOM size)
